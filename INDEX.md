## 📌 Key Information At a Glance

### **Project Name**
GenAI-FullStack - AI-powered interview preparation platform

### **In 30 Seconds**
Users upload resumes and job descriptions, and our system generates personalized interview reports with technical questions, behavioral questions, skill gaps analysis, and 7-day prep plans using Google Gemini AI.

### **Tech Stack Summary**
- **Frontend:** React 19 + Vite + React Router + Context API + SCSS
- **Backend:** Express.js 5 + Node.js
- **Database:** MongoDB + Mongoose 9
- **AI:** Google Gemini API 2.0-flash
- **Auth:** JWT + bcryptjs
- **File Processing:** pdf-parse + mammoth
- **Validation:** Zod

### **Key Features**
✅ User authentication (register, login, logout)
✅ Resume parsing (PDF & DOCX)
✅ AI-powered interview question generation
✅ Skill gap analysis
✅ 7-day preparation plan
✅ PDF report export
✅ Responsive UI

### **Architecture**
MVC pattern with service layer, Context API for frontend state, Mongoose ODM for database, JWT for authentication, schema-driven AI generation for guaranteed valid outputs.
